# Go rest api

## Requirements

- golang 1.11
- notify-tools: for inotifywait
- [godog](https://github.com/DATA-DOG/godog/)

This project uses the eperimental modules feature, so you will need to run:

```bash
export GO111MODULE=on
```

### Run the server

```bash
./dev-server.sh
```
